# ProArtist Africa - Quick Reference Card

## 🚀 Login
**URL:** `yoursite.com/wp-admin`

---

## 📝 Common Tasks

### Edit a Page
1. **Pages** → Find page → **Edit**
2. Make changes → **Update**

### Add New Brand
1. **Brands** → **Add New**
2. Add name, description, image
3. Fill stats → **Publish**

### Add New Job
1. **Jobs** → **Add New**
2. Add title, description, image
3. **Publish**

### Write Blog Post
1. **Posts** → **Add New**
2. Write content, add image
3. **Publish**

### Update Menu
1. **Appearance** → **Menus**
2. Add/remove items → **Save Menu**

## ⚠️ Important Reminders

- ✅ Always click **Update** or **Publish** to save changes
- ✅ Preview before publishing
- ✅ Use **Save Draft** if not ready
- ❌ Don't delete important code
- ❌ Don't share your password
