Manual

How to use the working sections: Blog and Analytics.

Blog

  • Open Blog to edit hero, blocks, and media.
  • Use Save Draft while working, then Publish Changes when ready.
  • Use Blog Drafts to reopen or delete drafts.
  • Use Blog Storage Audit to scan and clean unused files carefully.

Analytics

  • Open Analytics to read site views and page-level insights.
  • Use date filters to review a selected period.
  • Use Reset View Counters only when you intentionally want to clear stats.
