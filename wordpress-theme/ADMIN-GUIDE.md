# ProArtist Africa - Admin Guide

## Quick Start Guide for Website Managers

This guide will help you manage and update the ProArtist Africa website using WordPress admin.

---

## 1. First Time Setup

### Creating Your Admin Account

1. Go to your WordPress admin URL: `yoursite.com/wp-admin`
2. If you don't have an account yet, ask your developer to create one
3. Your developer will provide you with:
   - Username
   - Password
   - Admin URL

**Important:** Keep your password secure and don't share it!

---

## 2. Logging In

1. Go to: `yoursite.com/wp-admin`
2. Enter your username and password
3. Click "Log In"
4. You'll see the WordPress Dashboard

---

## 3. Common Tasks

### Editing Page Content

**To edit any page (Home, About, Investors, Brands, Jobs, Blog):**

1. Go to **Pages** → **All Pages** (left sidebar)
2. Find the page you want to edit
3. Click on the page name or hover and click **Edit**
4. Make your changes in the editor
5. Click **Update** (top right) to save

**Tip:** Use the "Visual" tab for easy editing (like Microsoft Word)

---

### Managing Brands

**To add a new brand:**

1. Go to **Brands** → **Add New** (left sidebar)
2. Enter the brand name as the title
3. Add description in the main editor
4. Click **Set featured image** (right sidebar) to add brand image
5. Scroll down to **Brand Statistics** section:
   - Enter number of Locations
   - Enter number of Customers
   - Enter Revenue amount
6. Click **Publish**

**To edit an existing brand:**

1. Go to **Brands** → **All Brands**
2. Click on the brand you want to edit
3. Make your changes
4. Click **Update**

---

### Managing Jobs

**To add a new job listing:**

1. Go to **Jobs** → **Add New** (left sidebar)
2. Enter the job title (e.g., "Professional Artists" or "Location Coordinators")
3. Add job description in the main editor
4. Add bullet points for benefits/responsibilities
5. Click **Set featured image** to add job image
6. Click **Publish**

**To edit or remove a job:**

1. Go to **Jobs** → **All Jobs**
2. Click on the job to edit, or hover and click **Trash** to remove

---

### Managing Blog Posts

**To add a new blog post:**

1. Go to **Posts** → **Add New** (left sidebar)
2. Enter the post title
3. Write your content in the editor
4. Click **Set featured image** to add a blog image
5. Add an excerpt (short description) - this shows on the blog listing page
6. Click **Publish**

**To edit existing posts:**

1. Go to **Posts** → **All Posts**
2. Click on the post you want to edit
3. Make changes and click **Update**

---

### Updating Contact Information

**To update email, phone, or social media links:**

1. Go to **Appearance** → **Theme Editor** (advanced) OR
2. Contact your developer for help (recommended)

**Note:** Contact info is in the footer. For safety, ask your developer to update it.

---

### Updating Navigation Menu

**To add/remove menu items:**

1. Go to **Appearance** → **Menus**
2. Select your menu (usually "Primary Menu")
3. Check/uncheck pages you want in the menu
4. Drag items to reorder
5. Click **Save Menu**

## 5. Best Practices

### Before Making Changes

1. **Preview first:** Click "Preview" before publishing
2. **Save drafts:** Use "Save Draft" if not ready to publish
3. **Backup:** Ask your developer about backups before major changes

### Image Guidelines

- **Recommended size:** 1200px wide for main images
- **File format:** JPG or PNG
- **File size:** Keep under 2MB for faster loading
- **Naming:** Use descriptive names (e.g., "mancave-interior.jpg")

### Content Tips

- Keep text clear and concise
- Use bullet points for lists
- Add images to make content engaging
- Proofread before publishing

## 6. Common Issues & Solutions

### "I can't see the changes on the website"
- **Solution:** Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
- Or wait a few minutes - sometimes changes take time to appear

### "I can't find where to edit something"
- **Solution:** Check if it's a Page, Post, Brand, or Job
- If unsure, contact your developer

### "The page looks broken after editing"
- **Solution:** Click "Undo" or contact your developer
- Don't delete important code sections

### "I forgot my password"
- **Solution:** Click "Lost your password?" on login page
- Or contact your developer to reset it

## 8. Getting Help

### When to Contact Your Developer

- Major design changes
- Adding new features
- Technical issues
- Security concerns
- Performance problems

### What Information to Provide

- What you were trying to do
- What happened (or didn't happen)
- Screenshot if possible
- Browser you're using

---

## 9. Security Tips

1. **Never share your password**
2. **Log out when done** (especially on shared computers)
3. **Use strong passwords** (mix of letters, numbers, symbols)
4. **Keep WordPress updated** (your developer handles this)
5. **Don't install plugins** without asking your developer
